# Голосовой помощник шустрик (имя можно изменить)
## Функции:
- Прогноз погоды
- Расписание
- Открытие программ
- Поиск в википедии
- Поздороваться
- Попрощаться

## Технологии (основные)
- python 3.12
- silero (генерация речи)
- vosk (распознавание речи)
- pandas
- pytorch

## Установка
- скачайте [python](https://www.python.org/) с официального сайта
- запустите setup.py
```bash
python setup.py
```
